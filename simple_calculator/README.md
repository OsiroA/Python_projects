# Simple Calculator

This is a simple calculator program written in Python. It allows you to perform basic arithmetic operations and other mathematical functions.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Exponentiation (**)
- Square Root (root)
- Logarithm (log)
- Sine (sin)
- Cosine (cos)
- Tangent (tan)

## Usage

1. Run the program using Python 3 (`python3 calculator.py`).
2. Enter the first number when prompted.
3. Enter the desired operation by choosing one of the available symbols (+, -, *, /, **, root, log, sin, cos, tan).
4. If applicable, enter the second number when prompted. If not applicable, leave it blank.
5. The program will calculate and display the result based on your inputs.

Please ensure that you have Python 3 installed on your system to run this program.

Note: The program validates the input to ensure it's a valid number and checks for valid operation symbols. It will display an error message if the input is invalid.

Feel free to modify and enhance the program as per your needs. Enjoy calculating!
